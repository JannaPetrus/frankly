# Claude Code — заметки по обучению

## Skills vs MCP vs Plugins vs Hooks

Важно понять разницу между четырьмя вещами — люди часто их путают:

- **Skills** — markdown-файлы с инструкциями. Занимают 30–50 токенов каждый, загружаются по требованию. Учат Claude как себя вести во время задачи.
- **MCP серверы** — живые подключения к внешним системам (GitHub, БД, браузер). Могут занимать 50k+ токенов при старте сессии.
- **Плагины** — бандлы из skills, команд, хуков и MCP конфигов в одном пакете. Устанавливаются как единое целое.
- **Хуки** — автоматические действия в определённые моменты (например, запустить линтер после изменения файла).
